# AI Art Gallery

A Pinterest-style gallery for AI-generated art.

## What this app will do

- Show AI art in a beautiful gallery grid
- Store artwork titles, prompts, models, and tags
- Let users browse, search, and save favorite images
- Eventually connect to Supabase for data and image storage
- Eventually deploy on Vercel

## Tech stack

- Next.js
- Supabase
- Vercel
- GitHub

## First goal

Build a simple homepage with a gallery of sample AI artwork.
